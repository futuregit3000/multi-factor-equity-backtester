# Resume Bullet Options

Use only claims you can explain comfortably in an interview.

## Two-bullet version

- Developed a Python multi-factor equity backtester combining momentum, value,
  quality, and low-volatility signals across a 50-stock U.S. large-cap universe;
  implemented point-in-time SEC filing alignment, monthly portfolio
  construction, naturally drifting weights, and transaction-cost modeling.
- Evaluated the strategy against SPY using CAGR, Sharpe, Sortino, drawdown,
  alpha/beta, VaR, Expected Shortfall, subperiod analysis, and cost sensitivity;
  the historical four-factor test produced a 17.4% CAGR, 0.90 Sharpe ratio, and
  -26.6% maximum drawdown from February 2016 through December 2025.

## One-bullet version

- Built a point-in-time Python equity-factor backtester using SEC EDGAR, FRED,
  and Yahoo Finance data; combined momentum, value, quality, and low-volatility
  signals with realistic monthly execution and transaction costs, producing a
  17.4% historical CAGR and 0.90 Sharpe ratio versus 15.5% and 0.77 for SPY.

## Safer version without headline performance numbers

- Designed and validated a reproducible multi-factor equity research pipeline
  in Python using point-in-time SEC fundamentals, monthly cross-sectional
  ranking, realistic rebalance timing, transaction costs, benchmark
  attribution, and robustness testing.

## Important wording guidance

Say **historical backtest**, not “the strategy returns 17.4%.”

Do not claim:

- Live trading performance
- Institutional production deployment
- Guaranteed alpha
- An unbiased historical universe
- Real investor capital under management
